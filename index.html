Data Cleaning & Feature Engineering

Raw data is rarely ready for analysis. Here is how the dataset was prepared using Python and Pandas.

Data Cleaning

  • Cabin Column: Dropped entirely due to 687 missing values (too many to impute reliably).
  • Age Imputation: Replaced missing ages (177 rows) with the median age specifically calculated by gender (Female: 27.0, Male: 29.0).
  • Embarked: Filled the 2 missing port values with the mode (most common port).

Feature Engineering

  • FamilySize: Created by adding SibSp (siblings/spouses), Parch (parents/children), plus 1 for the passenger.
  • IsAlone: A binary flag set to 1 if FamilySize equals 1.
  • FareClass & AgeGroup: Segmented continuous numerical variables into discrete bins to identify categorical trends.
  • Sex_num: Mapped 'female' to 1 and 'male' to 0 to process gender in the correlation matrix.
